反向传播

发表于2026-04-28|更新于2026-04-28|ai
|浏览量:

一、前言

1. 前向传播（Forward）

设一个 L 层神经网络：

$$
a^l = \sigma(a^{l-1} W^{l-1} + b^{l-1})
$$

其中：

  • $ (a^{l-1} = x) $（输入）
  • $ (a^l = \hat{y}) $（输出）

2. 损失函数

$$
L = \frac{1}{2}(\hat{y} - y)^2
$$

其中：

  • $ y $ 是训练数据的标签
  • $ \hat{y} $ 是前向传播的输出

3. 核心思想（链式法则）

反向传播的本质：

$$
\frac{\partial L}{\partial w} =
\frac{\partial L}{\partial a}
\cdot
\frac{\partial a}{\partial z}
\cdot
\frac{\partial z}{\partial w}
$$

二、反向传播的目标

反向传播的目标是计算每一层的权重和 bias 的梯度，从而向梯度下降的方向更新参数，所以需要计算损失函数对每一层权重和 bias 的偏导数：

定义 $ a^l $ 是第 l 层的输出，$ z^l = a^{l-1} w^{l-1} + b^{l-1} $ 是第 l 层的线性变换结果，$ \sigma(z^l) $ 是第 l 层的激活函数输出，那么损失函数可以写成：

$$
\begin{aligned}
L &= \frac{1}{2}(\hat{y} - y)^2 \
&= \frac{1}{2}(a^l - y)^2 \
&= \frac{1}{2}(\sigma(z^l) - y)^2 \
&= \frac{1}{2}(\sigma(a^{l-1} w^{l-1} + b^{l-1}) - y)^2
\end{aligned}
$$

则损失函数对权重和 bias 的偏导数可以通过链式法则计算：

$$
\begin{aligned}
\frac{\partial L}{\partial w^{l-1}}
&= (a^l - y) \cdot \frac{\partial a^l}{\partial w^{l-1}} \
&= (a^l - y) \cdot \sigma’(z^l) \frac{\partial z^l}{\partial w^{l-1}} \
&= (a^l - y) \cdot \sigma’(z^l) \cdot \frac{\partial (a^{l-1} w^{l-1} + b^{l-1})}{\partial w^{l-1}} \
&= (a^l - y) \cdot \sigma’(z^l) \cdot a^{l-1}
\end{aligned}
$$

$$
\begin{aligned}
\frac{\partial L}{\partial b^{l-1}}
&= (a^l - y) \cdot \frac{\partial a^l}{\partial b^{l-1}} \
&= (a^l - y) \cdot \sigma’(z^l) \cdot \frac{\partial (a^{l-1} w^{l-1} + b^{l-1})}{\partial b^{l-1}} \
&= (a^l - y) \cdot \sigma’(z^l) \cdot 1 \
&= (a^l - y) \cdot \sigma’(z^l)
\end{aligned}
$$

不难发现，每一层的权重依赖于当前层和上一层的输出，而 bias 只依赖于当前层的输出。

同时，如果不是最后一层，那么 y 并不是训练数据的标签，因此需要计算出该层的目标输出（也就是下一层的输入），为此，我们可以通过计算下一层的目标输入来反推当前层的目标输出：因为下一层的目标输入也可以通过梯度的方式计算出来，现在第 l 层的输入是 $ a^{l-1} $，输出是 $ a^l $，那么第 l-1 的目标输出就是第 l 层的输入向梯度的方向更新，即 $ a^{l-1} - \frac{\partial L}{\partial a^{l-1}} $，则损失函数为：

$$
\begin{aligned}
a^{l - 1} - y &= a^{l-1} - (a^{l-1} - \frac{\partial L}{\partial a^{l-1}}) \
&= \frac{\partial L}{\partial a^{l-1}} \
&= (a^l - y) \cdot \sigma’(z^l) \cdot w^{l-1}
\end{aligned}
$$

三、代码实现

这边使用的激活函数是 sigmoid 函数，其导数为 $ \sigma’(z) = \sigma(z)(1 - \sigma(z)) $，因此在计算梯度时需要用到当前层的输出值。

首先创建一个 back 的神经网络结构并初始化为 0：

其 ws 和 bs 分别存储每一层的权重和 bias 的梯度，其 val 存储每一层的输入的梯度，也就是上一层的目标输出与当前层输入的差值：

let mut back = nn_alloc(&nn.arch);
nn_zero(&mut back);

1. 前向传播

对于输入的每一个训练样本，我们需要将其复制到神经网络的输入矩阵中，然后执行前向传播：

// For each training example
for i in 0..ti.rows {
    mat_copy(nn.input(), &mat_row(&ti, i));
    nn_forward(nn);

2. 计算最后一层的梯度

首先把每一层的输入梯度初始化为 0，然后计算最后一层的输入梯度（也就是输出层的目标输出与当前输出的差值）：

for l in 0..nn.count + 1 {
    mat_fill(&mut back.vals[l], 0.0);
}

let target = mat_row(to, i);
// for output layer, calculate the derivative of the cost with respect to the output
for j in 0..back.vals[nn.count].cols {
    *back.vals[nn.count].at_mut(0, j) = nn.output().at(0, j) - target.at(j);
}

3. 当前层的输出以及上一层的输出梯度：

从最后一层开始，分别遍历该层的每一个激活值

  • a 是当前层的输出
  • gap 是当前层的输入梯度（也就是当前层的目标输入与上一层输出的差值）
  • dact 是当前层的激活函数的导数值（也就是当前层输出的导数值）
// For each layer
for l in (1..nn.count + 1).rev() {
    // For each activation in the layer
    for j in 0..nn.vals[l].cols {
        let a = *nn.vals[l].at(0, j);
        let gap = *back.vals[l].at(0, j);
        let dact = dactf(a, &nn.acts);
        *back.bs[l - 1].at_mut(0, j) += gap * dact;

4. 计算上一层的输出梯度

遍历上一层的每一个激活值，计算出上一层的输出梯度（也就是上一层的输出与当前层输入的差值）:

  • prea 是上一层的输出
  • w 是当前层的权重
// For each activation in the previous layer
for k in 0..nn.vals[l - 1].cols {
    let prea = *nn.vals[l - 1].at(0, k);
    let w = *nn.ws[l - 1].at(k, j);
    *back.ws[l - 1].at_mut(k, j) += gap * dact * prea;
    *back.vals[l - 1].at_mut(0, k) += gap * dact * w;
}

5. 平均权重和 bias

最后对每一层的权重和 bias 的梯度进行平均（除以训练样本的数量）：

for l in 0..nn.count {
    for i in 0..nn.ws[l].rows {
        for j in 0..nn.ws[l].cols {
            *back.ws[l].at_mut(i, j) /= ti.rows as f32;
        }
    }
}
for l in 0..nn.count {
    for i in 0..nn.bs[l].rows {
        for j in 0..nn.bs[l].cols {
            *back.bs[l].at_mut(i, j) /= ti.rows as f32;
        }
    }
}

四、总结

对于 nn，记录的是前向传播的结果，即每层的输入和输出；对于 back，记录的是反向传播的结果，即每层的权重和 bias 的梯度，以及每层输入的梯度（也就是上一层的目标输出与当前层输入的差值）。通过前向传播计算出每一层的输出，然后通过反向传播计算出每一层的权重和 bias 的梯度，最后对每一层的权重和 bias 的梯度进行平均，得到最终的梯度值。
